Commandes utilisateur






La commande a : Elle permet, avec une phrase d'entrée, d'enregistrer celle-ci à la suite dans l'EPPROM.





La commande d : Cette fonction permet de donner une phrase particulière à supprimer dans l'EPPORM.





la commande f : cette commande permet de connaître la quantité d'espace libre sur l'EEPROM.





La commande h : Ici, la commande h permet d'afficher toutes les commandes disponibles.





la commande l : A l'aide de cette commande, il est possible d'afficher toutes les phrases enregistrées sur l'EEPROM.





la commande r : Cela permet de selectionner puis jouer la phrase par défaut.
